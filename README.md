# xpected2

Static site for **xpected2.com** — a tactical engagement consultancy.

## Structure

```
xpected2/
├── index.html        ← landing page
├── styles.css        ← (currently empty; styles inline in index.html)
├── scripts.js        ← (currently empty)
├── images/
│   └── el-caballo.jpg
├── stats/
│   └── index.html    ← /stats/ subpage
├── .gitignore
└── README.md
```

## Deploy

Hosted on **Cloudflare Pages**. Two options:

1. **Direct upload:** Drag this folder into the Pages dashboard.
2. **Git workflow (recommended):** Push to GitHub, connect the repo to Pages, every push deploys automatically.

## Local preview

Open `index.html` in your browser — no build step needed.
